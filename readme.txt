== Description ==

Re-directs users to specific language landing pages that have been created by you.

Admin can generate pages according to language locals (i.e en-us, en-gb, se, de, etc...)
and this plugin will identify any browser loading the main page with a language preference and if a page matching exists will redirect users to that page.

== Installation ==

To install this plugin please use the "install" feature from the WordPress site.
